# Backend API

[API docs](/docs/api.md)

## Requirements
**Ruby**: 2.3.4 
**Rails**:5.1

## Setup
-  Installation
```
bundle install
```

-  Setup database
```
> bundle exec rake db:create
> bundle exec rake db:migrate
```

-  Setup Seeds (for development purposes only)
```
> bundle exec rake db:seeds
```

That's it.
